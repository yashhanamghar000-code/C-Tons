# 🌿 C-Tons Solution Website

A modern and responsive web application designed to showcase environmental impact solutions such as carbon credit services, I-REC certifications, and sustainability consulting.

---

## 📌 About the Project

C-Tons Solution is a platform built to help businesses understand and adopt sustainable practices. It provides structured information about various environmental services and solutions through an intuitive and visually appealing interface.

---

## ✨ Features

- Fully responsive design (mobile + desktop)
- Clean and modern UI
- Smooth navigation between pages
- Modular and reusable components
- Multiple service-based pages

---

## 🧩 Pages Included

- Home Page
- About Page
- Services Page  
- Project Developers Page  
- Corporate Companies Page  
- Voluntary Market Page  
- Compliance Market Page  
- Carbon Footprint Accounting Page  
- Energy Audit Page  
- ESG Reporting Page  

---

## 🛠 Tech Stack

- React (with TypeScript)
- Tailwind CSS
- React Router DOM
- Vite

---

## 📁 Project Structure


src/
├── assets/ # Images used in the project
├── components/ # Reusable UI components (Navbar, Footer, etc.)
├── pages/ # All website pages
├── App.tsx # Routing configuration
├── main.tsx # Entry point


---

## ⚙️ How to Run the Project

1. Clone the repository

git clone https://github.com/yashhanamghar000-code/C-Tons.git


2. Open the project folder

cd C-Tons


3. Install dependencies

npm install


4. Start development server

npm run dev
